# 内容类型和列表的字段集合

## 公共属性

所有字段类型共享以下属性：

| 属性 | 外观 | 验证规则 | 必填 |
|------|------|----------|------|
| 字段名称 | `el-input` | 必填，最大 64 字符 | 是 |
| 字段键 (key) | `el-input` | 必填，仅小写字母/数字/下划线，唯一，不可更改 | 是 |
| 是否必填 | `el-switch` | — | 否 |
| 默认值 | `el-input` | 类型需与字段类型匹配 | 否 |
| 帮助文本 | `el-input` | 最大 256 字符 | 否 |
| 占位符 | `el-input` | 最大 128 字符 | 否 |

## 表单


### 字段类型

字段的类型在新增提交之后，不可更改


#### 文本（长文本）

| 属性 | 外观 | 验证规则 |
|------|------|----------|
| 类型 | `el-radio-group`：单行 / 多行 / 长文本，默认单行 | 必填 |
| 最小长度 | `el-input-number` `:min="0"` `:step="1"` | 整数，必须 ≤ 最大长度；类型为"长文本"时隐藏 |
| 最大长度 | `el-input-number` `:min="0"` `:step="1"` | 整数，必须 ≥ 最小长度；类型为"长文本"时隐藏 |
| 行数 | `el-input-number` `:min="1"` `:max="20"` `:step="1"` | 整数，类型为"多行时"默认值为 3， 类型为"长文本"时默认值为 10，类型为"单行"时隐藏 |
| 支持富文本 | `el-switch` | 默认关闭；类型为"长文本"时显示 |

字段在表单中的外观：

| 类型 | 表单组件 |
|------|----------|
| 单行 | `el-input` |
| 多行 | `el-input type="textarea"` |
| 长文本（富文本关闭） | `el-input type="textarea"` `:rows="行数"` |
| 长文本（富文本开启） | 富文本编辑器（TinyMCE / Quill） |

当类型切换时，之前类型特有的属性值保留但不参与验证；切换回该类型时恢复。

**表格呈现**：纯文本截断显示（长文本富文本需去除 HTML 标签），超出省略号

**JSONB 存储格式**：`"field_key": "string value"`（富文本存储为 HTML 字符串）

---

#### 数字

| 属性 | 外观 | 验证规则 |
|------|------|----------|
| 最小值 | `el-input-number` | 数字类型，必须 ≤ 最大值 |
| 最大值 | `el-input-number` | 数字类型，必须 ≥ 最小值 |
| 精度 | `el-input-number` `:min="0"` `:max="10"` `:step="1"` | 整数，0~10。0=整数，2=两位小数 |

- 精度与最小/最大值的联动：最小值和最大值的小数位数不能超过精度。例如精度=2，最小值 "6.123" 不合法
- 字段在表单中的外观：`el-input-number` `:precision="精度"`

**表格呈现**：右对齐，按精度格式化；整数不显示小数点

**JSONB 存储格式**：`"field_key": 123` 或 `"field_key": 12.50`（存储为数字，非字符串）

---

#### 选项

| 属性 | 外观 | 验证规则 |
|------|------|----------|
| 选项列表 | `el-input type="textarea"` `:rows="5"` | 必填，每行一个选项，至少 2 个选项 |
| 类型 | `el-radio-group`：单选 / 多选，默认单选 | 必填 |
| 呈现 | `el-radio-group`，见下方规则 | 必填（多选时） |

呈现规则：

| 类型 | 可用呈现方式 | 默认值 |
|------|-------------|--------|
| 单选 | 单选按钮组 (`el-radio-group`) / 下拉列表 (`el-select`) | 单选按钮组 |
| 多选 | 复选框 (`el-checkbox-group`) / 下拉列表 (`el-select` multiple) | 下拉列表 |

- 当类型切换时，"呈现"值应重置为该类型的默认值
- 字段在表单中的外观：根据"类型"和"呈现"组合决定（见上表）

**表格呈现**：已选选项以标签形式展示，超过 3 个折叠为 "+N"

**JSONB 存储格式**：
- 单选：`"field_key": "选项A"`
- 多选：`"field_key": ["选项A", "选项C"]`

---

#### 日期

| 属性 | 外观 | 验证规则 |
|------|------|----------|
| 日期格式 | `el-select`：日期 / 日期时间 | 必填，默认"日期" |
| 最早日期 | `el-date-picker` | 可选，选中的值必须 ≥ 最早日期 |
| 最晚日期 | `el-date-picker` | 可选，选中的值必须 ≤ 最晚日期，必须 ≥ 最早日期 |

- 字段在表单中的外观：`el-date-picker`，根据"日期格式"切换 `type="date"` 或 `type="datetime"`
- 最早/最晚日期的 picker 不需要时间部分（始终是 date 类型）

**表格呈现**：格式化显示，日期格式 "YYYY-MM-DD"，日期时间格式 "YYYY-MM-DD HH:mm:ss"

**JSONB 存储格式**：`"field_key": "2026-05-17"` 或 `"field_key": "2026-05-17T14:30:00"`（ISO 8601 字符串）

---

#### 布尔

| 属性 | 外观 | 验证规则 |
|------|------|----------|
| 呈现 | `el-radio-group`：开关 / 单选按钮，默认开关 | 必填 |
| 显示的值 | `el-input`：分别对应 `true` 和 `false`, 默认值 分别为 `是` 和 `否` | 必填 |

- 字段在表单中的外观：开关为 `el-switch`，单选按钮为 `el-radio-group`
- 默认值为 `false`

**表格呈现**：显示`true` 和 `false` 分别对应的值

**JSONB 存储格式**：`"field_key": true` 或 `"field_key": false`（存储为布尔值）

---

#### 自动编号

自动编号在创建记录时由系统生成，表单中只读展示，用户不可手动编辑。

| 属性 | 外观 | 验证规则 |
|------|------|----------|
| 生成规则 | `el-select`：前缀+序号 / 日期+序号 / 纯序号 / 随机码 | 必填，默认"前缀+序号" |
| 前缀 | `el-input` | 最大 20 字符，生成规则为"纯序号"或"随机码"时隐藏 |
| 起始序号 | `el-input-number` `:min="1"` `:step="1"` | 整数 ≥ 1，默认 1；生成规则为"日期+序号"或"随机码"时隐藏 |
| 序号位数 | `el-input-number` `:min="2"` `:max="12"` `:step="1"` | 整数 2~12，默认 5。不足位数左侧补零；生成规则为"随机码"时隐藏 |
| 日期格式 | `el-select`：YYYYMMDD / YYMMDD / YYYYMM / YYMM | 生成规则为"日期+序号"时显示，默认 YYYYMMDD |
| 随机码长度 | `el-input-number` `:min="4"` `:max="32"` `:step="1"` | 整数 4~32，默认 8；生成规则为"随机码"时显示 |
| 随机码字符集 | `el-checkbox-group`：大写字母 / 小写字母 / 数字 | 至少选一项，默认全选；生成规则为"随机码"时显示 |

生成示例（假设序号位数=5，起始序号=1，前缀="CASE"）：

| 生成规则 | 示例 |
|----------|------|
| 前缀+序号 | `CASE-00001` |
| 日期+序号 | `20260517-00001` |
| 纯序号 | `00001` |
| 随机码 | `aB3xK9mQ` |

- 序号按列表维度自增，每个列表独立计数
- 字段在表单中的外观：`el-input` `:disabled="true"`（只读，创建时展示生成预览，提交后写入实际值）
- "公共属性"中的"是否必填"和"默认值"对本类型无效，始终必填且由系统生成

**表格呈现**：纯文本，等宽字体显示

**JSONB 存储格式**：`"field_key": "CASE-00001"`（存储为字符串）

#### 附件

| 属性 | 外观 | 验证规则 |
|------|------|----------|
| 最大文件数 | `el-input-number` `:min="1"` `:max="50"` `:step="1"` | 整数 |
| 允许的文件类型 | `el-select` multiple：图片 / 文档 / 视频 / 音频 / 全部 | 至少选一项，默认"全部" |
| 单文件最大大小 (MB) | `el-input-number` `:min="1"` `:max="100"` `:step="1"` | 整数，默认 10 |

- 字段在表单中的外观：`el-upload`，`multiple` 由最大文件数 > 1 决定，`accept` 由允许的文件类型决定

**表格呈现**：显示文件数量和总大小，如 "3 个文件 (2.5 MB)"

**JSONB 存储格式**：`"field_key": [{"name": "file.pdf", "url": "/api/files/xxx", "size": 1024000, "type": "application/pdf"}]`

#### 引用

| 属性 | 外观 | 验证规则 |
|------|------|----------|
| 引用的内容类型 | `el-select`，列出当前应用下所有内容类型 | 必填 |
| 引用方式 | `el-radio-group`：单选 / 多选，默认单选 | 必填 |
| 显示字段 | `el-select`，目标内容类型的字段列表 | 必填，决定下拉列表和表格中显示的文本 |
| 呈现 | `el-radio-group`：下拉列表 / 树形选择，默认下拉列表 | 必填；选择"树形选择"时，目标内容类型必须支持树结构（有自引用字段） |

当呈现为"树形选择"时，额外显示以下配置：

| 属性 | 外观 | 验证规则 |
|------|------|----------|
| 父节点字段 | `el-select`，目标内容类型中引用自身的字段 | 必填，决定树结构的父子关系 |
| 选择策略 | `el-radio-group`：仅叶子 / 仅父节点 / 全部可选，默认"仅叶子" | 引用方式为"多选"时显示 |
| 展开方式 | `el-radio-group`：点击 / 悬停，默认"点击" | — |
| 可搜索 | `el-switch` | 默认开启 |
| 展开所有节点 | `el-switch` | 默认关闭，开启后初始展开所有层级 |

- 当"引用的内容类型"变化时，需重新加载"显示字段"和"父节点字段"的可选列表；若新内容类型不支持树结构，"呈现"自动重置为"下拉列表"
- 字段在表单中的外观：

| 呈现 | 引用方式 | 表单组件 |
|------|----------|----------|
| 下拉列表 | 单选 | `el-select`（搜索式下拉，从目标列表加载记录） |
| 下拉列表 | 多选 | `el-select` multiple |
| 树形选择 | 单选 | `el-tree-select` `:check-strictly="false"` |
| 树形选择 | 多选 | `el-tree-select` multiple，`check-strictly` 由选择策略决定 |

**表格呈现**：显示被引用记录的"显示字段"值（单选用文本链接，多选用标签）。值可点击，跳转到被引用记录所在列表的对应记录详情页

**JSONB 存储格式**：
- 单选：`"field_key": {"id": "uuid", "_display": "显示的文本值"}`
- 多选：`"field_key": [{"id": "uuid1", "_display": "文本1"}, {"id": "uuid2", "_display": "文本2"}]`

#### 计算值

计算值由表达式动态生成，不存储数据，只在读取记录时实时计算并展示。

| 属性 | 外观 | 验证规则 |
|------|------|----------|
| 表达式 | `el-input type="textarea"` `:rows="3"` | 必填，语法 `{field_key}` 引用同类型字段，可混合字面文本 |

表达式语法：

| 语法 | 说明 | 示例 |
|------|------|------|
| `{field_key}` | 引用当前记录中其他字段的值 | `{first_name}` |
| 字面文本 | 直接拼接的固定字符串 | `-`、`(`、`)` |
| `\n` | 换行符 | `{province}\n{city}` |

示例（假设字段 `first_name="张"`, `last_name="三"`）：

| 表达式 | 计算结果 |
|--------|----------|
| `{last_name}{first_name}` | `三张` |
| `{last_name}-{first_name}` | `三-张` |
| `{province}-{city}-{district}` | `广东省-深圳市-南山区` |

- 被引用的字段必须在同一内容类型中已存在，不能引用自身或其他计算值字段（避免循环依赖）
- 引用的字段值变化时，计算值自动更新
- 字段在表单中的外观：`el-input` `:disabled="true"`（只读展示计算结果，不在表单中存储）
- "公共属性"中的"是否必填"和"默认值"对本类型无效

**表格呈现**：纯文本，支持换行符渲染为 `<br>` 或多行显示

**JSONB 存储格式**：不存储，仅在读取时计算。前端和导出数据中包含计算结果，但写入时忽略该字段

---

#### 数组

数组字段存储一组子记录，每条子记录遵循指定内容类型的字段结构。在表单中呈现为内嵌表格，支持增删改查、批量操作、从其他列表导入和拖拽排序。

| 属性 | 外观 | 验证规则 |
|------|------|----------|
| 引用的内容类型 | `el-select`，列出当前应用下所有内容类型 | 必填，决定子记录的字段结构 |
| 最小条目数 | `el-input-number` `:min="0"` `:step="1"` | 整数，必须 ≤ 最大条目数 |
| 最大条目数 | `el-input-number` `:min="1"` `:step="1"` | 整数，必须 ≥ 最小条目数 |
| 排序字段 | `el-select`，目标内容类型的数字字段列表 | 可选，指定后子记录按该字段值排序；拖拽排序时自动更新该字段值 |

操作权限（均默认开启）：

| 属性 | 外观 | 说明 |
|------|------|------|
| 允许新增 | `el-switch` | 内部新增按钮 + 弹出表单 |
| 允许编辑 | `el-switch` | 行内操作列的编辑按钮 |
| 允许删除 | `el-switch` | 行内操作列的删除按钮（单个删除） |
| 允许批量删除 | `el-switch` | 工具栏批量删除按钮，依赖表格首列多选 |
| 允许排序 | `el-switch` | 拖拽手柄，调整子记录顺序 |
| 允许从其他列表添加 | `el-switch` | 工具栏导入按钮，从其他列表多选导入 |

当"允许从其他列表添加"开启时，额外显示：

| 属性 | 外观 | 验证规则 |
|------|------|----------|
| 来源列表 | `el-select`，当前应用下基于同一内容类型创建的列表 | 必填 |
| 去重字段 | `el-select`，目标内容类型的字段列表 | 必填，导入时根据此字段判断当前数组中是否已存在，已存在的记录在弹出列表中禁用选择 |

---

**表单布局**（上下结构）：

```
┌─────────────────────────────────────────────┐
│  [+ 新增]  [从其他列表添加]  [批量删除]       │  ← 工具栏
├────┬──────────┬──────────┬──────┬───────────┤
│ ☐  │ 字段A   │ 字段B    │ ...  │ 操作       │  ← 表头
├────┼──────────┼──────────┼──────┼───────────┤
│ ☐  │ 值1     │ 值2      │ ...  │ ⋮ 编辑 删除│  ← 数据行
│ ☐  │ 值3     │ 值4      │ ...  │ ⋮ 编辑 删除│
│ ⋮  │ ⋮       │ ⋮        │ ⋮    │ ⋮         │
└────┴──────────┴──────────┴──────┴───────────┘
```

- 表格列：首列为多选框，中间为引用的内容类型字段（不含计算值字段），最后一列为操作列（编辑、删除，由权限控制是否显示）
- 排序开启时，首列左侧额外显示拖拽手柄 `⋮⋮`
- 新增/编辑：弹出 `el-dialog`，内嵌 `DynamicForm`，字段由引用的内容类型定义
- 从其他列表添加：弹出 `el-dialog`，内嵌一个带多选和搜索的表格，已存在于当前数组中的记录（按去重字段匹配）其复选框置灰 `:disabled`，确认后将选中记录追加到当前数组末尾

**表格呈现**：显示条目数量徽章，如 "5 条记录"

**JSONB 存储格式**：
```json
"field_key": [
  {"id": "uuid-sub-1", "data": {"name": "子记录1", "price": 100}},
  {"id": "uuid-sub-2", "data": {"name": "子记录2", "price": 200}}
]
```
每条子记录有独立 UUID，`data` 内存储子记录的各字段值。

## 字段呈现汇总

字段添加到列表后，在表单和表格中的呈现方式如下：

| 字段类型 | 表单 (Form) | 表格 (Table) |
|----------|-------------|--------------|
| 文本（单行） | `el-input` | 纯文本截断，超出省略号 |
| 文本（多行） | `el-input type="textarea"` | 同单行 |
| 文本（长文本，纯文本） | `el-input type="textarea"` `:rows="行数"` | 纯文本截断 |
| 文本（长文本，富文本） | 富文本编辑器 | 去除 HTML 标签后截断 |
| 数字 | `el-input-number` `:precision="精度"` | 右对齐，按精度格式化 |
| 选项（单选+按钮组） | `el-radio-group` | 标签 |
| 选项（单选+下拉） | `el-select` | 标签 |
| 选项（多选+复选框） | `el-checkbox-group` | 标签，超过 3 个折叠 "+N" |
| 选项（多选+下拉） | `el-select` `multiple` | 同上 |
| 日期（日期） | `el-date-picker type="date"` | `YYYY-MM-DD` |
| 日期（日期时间） | `el-date-picker type="datetime"` | `YYYY-MM-DD HH:mm` |
| 布尔（开关） | `el-switch` | 显示 `true` / `false` 对应的值 |
| 布尔（单选按钮） | `el-radio-group` | 同上 |
| 自动编号（前缀+序号） | `el-input` `:disabled` | 等宽字体，如 `CASE-00001` |
| 自动编号（日期+序号） | `el-input` `:disabled` | 等宽字体，如 `20260517-00001` |
| 自动编号（纯序号） | `el-input` `:disabled` | 等宽字体，如 `00001` |
| 自动编号（随机码） | `el-input` `:disabled` | 等宽字体，如 `aB3xK9mQ` |
| 附件 | `el-upload` | 文件数量 + 总大小 |
| 引用（单选+下拉） | `el-select`（搜索式） | 可点击的文本链接，跳转到被引用记录详情页 |
| 引用（单选+树形） | `el-tree-select` | 同上 |
| 引用（多选+下拉） | `el-select` `multiple` | 可点击的标签，点击跳转到被引用记录详情页 |
| 引用（多选+树形） | `el-tree-select` `multiple` | 同上 |
| 计算值 | `el-input` `:disabled`（只读展示） | 纯文本，换行符渲染为 `<br>` |
| 数组 | 内嵌表格 + 工具栏（新增/导入/批量删除） | 条目数量徽章，如 "5 条记录" |

## 字段联动规则

- **选项**：切换"类型"时，"呈现"重置为默认值
- **引用**：切换"引用的内容类型"时，清空"显示字段"和"父节点字段"的选择；若新内容类型不支持树结构，"呈现"重置为"下拉列表"。切换"引用方式"时，根据方式显示/隐藏"选择策略"
- **数字**：切换"最小值"时，如果"最大值" < 新的"最小值"，则"最大值"同步更新为新的"最小值"；反之亦然
- **文本**：切换"类型"时，根据类型显示/隐藏对应的属性（最小长度/最大长度、行数、富文本）。最小长度 ↔ 最大长度遵循极值联动规则
- **自动编号**：切换"生成规则"时，根据规则显示/隐藏对应的属性（前缀、起始序号、序号位数、日期格式、随机码长度、随机码字符集）
- **计算值**：表达式中引用的字段被删除或重命名时，需提示用户更新表达式。不能引用自身或其他计算值字段，保存时校验循环依赖
- **数组**：切换"引用的内容类型"时，清空"排序字段""来源列表""去重字段"的选择。"允许从其他列表添加"关闭时隐藏来源列表和去重字段。最小条目数 ↔ 最大条目数遵循极值联动规则
